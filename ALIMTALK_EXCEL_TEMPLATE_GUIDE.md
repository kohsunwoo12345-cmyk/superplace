# 알림톡 대량 발송 엑셀 템플릿 가이드

## 📋 엑셀 파일 형식

엑셀 파일은 다음 3개의 필수 컬럼을 포함해야 합니다:

| 학생이메일 | 학부모이름 | 학부모연락처 |
|-----------|-----------|-------------|
| student001@academy.com | 김영희 | 010-1234-5678 |
| student002@academy.com | 이철수 | 010-2345-6789 |
| student003@academy.com | 박민수 | 010-3456-7890 |

### 컬럼 설명

#### 1. 학생이메일 (필수)
- **다른 이름**: `studentEmail`, `이메일`, `email`
- **형식**: 학생의 고유 이메일 주소
- **예시**: `student001@academy.com`, `hongkildong@school.com`
- **용도**: 학생을 식별하고 해당 학생의 최신 랜딩페이지를 자동으로 가져옵니다

#### 2. 학부모이름 (필수)
- **다른 이름**: `parentName`, `이름`, `name`
- **형식**: 학부모 이름
- **예시**: `김영희`, `이철수`
- **용도**: 알림톡 메시지의 `#{name}` 변수에 치환됩니다

#### 3. 학부모연락처 (필수)
- **다른 이름**: `parentPhone`, `전화번호`, `phone`, `휴대폰`
- **형식**: 숫자만 입력하거나 하이픈 포함 (자동으로 숫자만 추출)
- **예시**: `01012345678`, `010-1234-5678`
- **용도**: 알림톡 수신자 전화번호

## 🔄 자동 처리 과정

1. **엑셀 업로드**: 위 형식의 엑셀 파일을 업로드합니다
2. **학생 조회**: 각 학생이메일로 DB에서 학생 정보를 조회합니다
3. **최신 리포트 조회**: 각 학생의 가장 최근에 생성된 랜딩페이지를 자동으로 가져옵니다
4. **URL 자동 생성**: 학생 ID를 포함한 고유 URL이 자동으로 생성됩니다
5. **변수 치환**: 템플릿의 `#{name}`과 `#{url}` 변수가 자동으로 치환됩니다
6. **발송 준비 완료**: "발송" 버튼을 클릭하면 Solapi API를 통해 대량 발송됩니다

## 📊 처리 결과

업로드 후 다음과 같은 통계를 확인할 수 있습니다:

- ✅ **처리 완료**: 정상적으로 처리된 수신자 수
- ⚠️ **학생을 찾을 수 없음**: 해당 이메일의 학생이 DB에 없음
- ⚠️ **리포트가 없음**: 학생은 있지만 생성된 랜딩페이지가 없음
- ⚠️ **정보 누락**: 필수 컬럼(이메일, 연락처)이 비어있음
- ❌ **오류 발생**: 처리 중 오류가 발생한 경우

[학습 안내]

### 실제 발송 예시

**입력 데이터**:
- 학생이메일: `hongkildong@academy.com`
- 학부모이름: `홍길동`
- 학부모연락처: `010-1234-5678`

**발송 메시지**:
```
[학습 안내]

안녕하세요, 홍길동 학생 학부모님
꾸메땅학원입니다.

오늘 준비된 맞춤형 학습 페이지 안내드립니다.
아래 링크를 클릭하여 이번달 리포트를 확인해 주세요!

■ 학습 페이지: https://superplacestudy.pages.dev/landing/page123?studentId=student-001&ref=1234567890_0

※ 본 메시지는 수신 동의하신 분들께 발송되는 학습 안내 정보입니다.
```

## ⚙️ DB 테이블 구조

### 새로 추가된 테이블

1. **parents**: 학부모 정보
2. **student_parents**: 학생-학부모 연결 (N:M 관계)
3. **student_reports**: 학생별 리포트(랜딩페이지) 저장
4. **alimtalk_send_history**: 알림톡 발송 이력
5. **alimtalk_recipients**: 알림톡 수신자 상세 정보

### 주요 관계

```
students (학생)
  ├─ student_parents ─→ parents (학부모)
  └─ student_reports ─→ landing_pages (랜딩페이지)

알림톡 발송:
alimtalk_send_history (발송 이력)
  └─ alimtalk_recipients (수신자 상세)
       ├─ students (학생 연결)
       └─ parents (학부모 연결)
```

## 📝 주의사항

1. **학생 이메일은 고유해야 합니다**: 중복된 이메일이 있으면 첫 번째 학생만 조회됩니다
2. **랜딩페이지가 없는 학생**: 리포트를 먼저 생성해야 발송이 가능합니다
3. **전화번호 형식**: 하이픈이 있어도 자동으로 제거되므로 어떤 형식이든 가능합니다
4. **엑셀 컬럼명**: 한글 또는 영문 모두 지원합니다 (위 목록 참조)

## 🚀 사용 방법

1. 알림톡 발송 페이지 접속
2. 채널 선택
3. 템플릿 선택 (기본 템플릿 3 - 학습 안내 추천)
4. "엑셀" 탭 클릭
5. 위 형식의 엑셀 파일 업로드
6. 자동 처리 완료 후 통계 확인
7. 미리보기에서 메시지 확인
8. "X명에게 발송" 버튼 클릭
