### 2. **가격 계산 로직 개선**
**수정 전:**
```typescript
// 잘못된 계산: 학생수 × 개월수 × 단가
return count * months * basePrice;
```

**수정 후:**
```typescript
// 올바른 계산: (기본가격 + 학생당가격 × 학생수) × 개월수
const basePrice = product.monthlyPrice || product.price || 0;
const pricePerStudent = product.pricePerStudent || 0;
return (basePrice + (pricePerStudent * count)) * months;
```

**예시:**
- 기본가격: 50,000원/월
- 학생당가격: 10,000원/월
- 학생수: 20명
- 이용기간: 3개월

**계산:**
```
(50,000 + 10,000 × 20) × 3 = (50,000 + 200,000) × 3 = 750,000원
```
