### 1. 헤더 레이아웃 대폭 개선 ✅

#### **Before (이전)**
```
[로고] [검색...] [AI 쇼핑몰] [홈페이지] [알림] [사용자 정보]
```

#### **After (개선 후)**
```
[로고] [AI 쇼핑몰] [홈페이지] ───[넓은 검색창]─── [알림] [사용자 정보]
```

#### **주요 변경사항**
✅ **AI 쇼핑몰 & 홈페이지 버튼** → 왼쪽으로 이동 (로고 바로 옆)
- 더 눈에 띄고 접근하기 쉬운 위치
- 아이콘 + 텍스트로 명확한 표시
- AI 쇼핑몰: 파란색→보라색 그라데이션 + 애니메이션
- 홈페이지: 초록색→청록색 그라데이션

✅ **검색창 대폭 확장**
- 기존: 작은 버튼 형태 (`px-4 py-2`)
- 개선: 최대 2xl 너비 (`max-w-2xl`) + 실제 입력창
- `xl` 화면 이상에서 표시
- 플레이스홀더: "학생, 교사, 수업 검색..."
- 포커스 시 흰색 배경 + 파란색 링 효과

✅ **사용자 정보** → 오른쪽 끝으로 이동
- 알림 아이콘 옆에 배치
- 구분선으로 시각적 분리
- 사용자 이름 + 이메일 표시
- 로그아웃 버튼 (호버 시 빨간색)

#### **반응형 디자인**
- **Desktop (xl+)**: 전체 레이아웃 표시 (검색창 포함)
- **Tablet (lg)**: 버튼만 표시 (검색창 숨김)
- **Mobile**: 햄버거 메뉴 + 최소 요소만

### 4. 학원 권한 설정 컴포넌트 신규 추가 ✅

#### **파일 경로**
```
src/components/AcademyPermissionSettings.tsx
```

#### **주요 기능**

##### **🕐 일일 사용 한도 설정**
- AI 요청 횟수 제한 (0 ~ 10,000)
- 실시간 입력 필드 (숫자 타입)
- 큰 숫자로 현재 설정값 표시
- 권장 설정 안내:
  - Basic 플랜: 50-100회/일
  - Standard 플랜: 200-500회/일
  - Premium 플랜: 무제한 (10,000회)

##### **🧠 AI 기반 분석 기능**

###### **1. AI 기반 역량 분석** (`aiAnalysisEnabled`)
- **아이콘**: `Zap` (보라색)
- **설명**: 학생의 학습 패턴을 분석하여 강점과 약점을 파악합니다
- **Switch**: 활성화/비활성화 토글

###### **2. 개념 분석 실행** (`conceptAnalysisEnabled`)
- **아이콘**: `BookOpen` (인디고색)
- **설명**: 학습한 개념의 이해도를 분석하고 취약점을 찾습니다
- **Switch**: 활성화/비활성화 토글

##### **📚 학습 관리 기능**

###### **3. 숙제 체크 기능** (`homeworkCheckEnabled`)
- **아이콘**: `Settings` (초록색)
- **설명**: AI가 학생의 숙제를 자동으로 검사하고 피드백을 제공합니다
- **Switch**: 활성화/비활성화 토글

###### **4. 출석 관리** (`attendanceTrackingEnabled`)
- **아이콘**: `Clock` (파란색)
- **설명**: 학생 출석을 자동으로 추적하고 통계를 생성합니다
- **Switch**: 활성화/비활성화 토글

###### **5. 학습 리포트 생성** (`reportGenerationEnabled`)
- **아이콘**: `TrendingUp` (주황색)
- **설명**: 학생별 학습 리포트를 자동으로 생성하고 학부모에게 전송합니다
- **Switch**: 활성화/비활성화 토글

##### **📢 마케팅 & 알림 기능**

###### **6. SMS 문자 발송** (`smsNotificationEnabled`)
- **아이콘**: `Settings` (청록색)
- **설명**: 학부모에게 학습 리포트 및 공지사항을 문자로 발송합니다
- **Switch**: 활성화/비활성화 토글

###### **7. 랜딩 페이지 생성** (`landingPageEnabled`)
- **아이콘**: `Settings` (인디고색)
- **설명**: 학생별 학습 데이터를 학부모에게 공유할 랜딩 페이지를 생성합니다
- **Switch**: 활성화/비활성화 토글
