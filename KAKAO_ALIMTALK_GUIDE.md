# 카카오 알림톡 템플릿 관리 시스템

SUPER PLACE의 카카오 알림톡 템플릿 관리 시스템입니다. 솔라피(Solapi)를 통해 카카오 알림톡 템플릿을 생성, 관리, 발송할 수 있습니다.

## 📋 주요 기능

### 1. 템플릿 관리
- ✅ **템플릿 생성**: 카카오 알림톡 템플릿 생성
- ✅ **템플릿 수정**: 대기/반려 상태 템플릿 수정
- ✅ **템플릿 삭제**: 템플릿 삭제 (검수중 제외)
- ✅ **템플릿 조회**: 목록 및 상세 조회
- ✅ **검수 요청**: 템플릿 검수 요청 및 취소
- ✅ **상태 관리**: 대기, 검수중, 승인, 반려 상태 관리

### 2. 실시간 미리보기
- ✅ **실제 카카오톡 UI**: 실제 카카오톡 화면과 동일한 미리보기
- ✅ **4가지 메시지 유형 지원**:
  - 기본형 (BA)
  - 부가정보형 (EX)
  - 광고추가형 (AD)
  - 복합형 (MI)
- ✅ **강조 유형 지원**:
  - 선택안함 (NONE)
  - 강조표기형 (TEXT)
  - 이미지형 (IMAGE)
  - 아이템리스트형 (ITEM_LIST)

### 3. 버튼 및 바로연결
- ✅ **버튼**: 최대 5개 (웹링크, 앱링크, 배송조회, 봇키워드 등)
- ✅ **바로연결**: 최대 10개 (웹링크, 앱링크, 봇전환 등)
- ✅ **다양한 버튼 타입**:
  - WL: 웹링크
  - AL: 앱링크
  - DS: 배송조회
  - BK: 봇키워드
  - MD: 메시지전달
  - BC: 상담톡전환
  - BT: 봇전환
  - AC: 채널추가

### 4. 알림톡 발송
- ✅ **검수 완료 템플릿만 발송 가능**
- ✅ **템플릿 내용 변경 불가**: 변수만 치환하여 발송
- ✅ **변수 자동 추출**: `#{변수명}` 형식 자동 인식
- ✅ **발송 전 미리보기**: 변수 치환된 실제 발송 내용 확인

## 🚀 시작하기

### 1. 솔라피 계정 생성 및 API 키 발급

1. [솔라피](https://solapi.com) 회원가입
2. 카카오 채널 연동
3. API 키 발급:
   - Dashboard > API Keys
   - API Key와 API Secret 복사

### 2. 환경 변수 설정

`.env` 파일에 다음 내용을 추가하세요:

```env
# Solapi (카카오 알림톡)
SOLAPI_API_KEY="your-solapi-api-key"
SOLAPI_API_SECRET="your-solapi-api-secret"
SOLAPI_CHANNEL_ID="@채널ID" # 카카오 채널 pfId
SOLAPI_SENDER_NUMBER="01012345678" # 발신 번호 (하이픈 없이)
```

### 3. 카카오 채널 확인

솔라피 대시보드에서 카카오 채널 ID(pfId)를 확인하세요:
- Dashboard > 카카오 채널 관리
- 채널 ID는 `@` 로 시작합니다 (예: `@mycompany`)

## 📱 사용 방법

### 템플릿 생성

1. **대시보드 접속**
   ```
   /dashboard/kakao/templates
   ```

2. **"템플릿 생성" 버튼 클릭**

3. **템플릿 정보 입력**:
   - 템플릿 이름 (중복 불가)
   - 채널 ID (pfId)
   - 카테고리 선택
   - 템플릿 내용
   - 메시지 유형 선택
   - 강조 유형 선택 (선택사항)
   - 버튼 추가 (선택사항)
   - 바로연결 추가 (선택사항)

4. **실시간 미리보기 확인**
   - 오른쪽에 실제 카카오톡 화면 표시
   - 입력하는 즉시 반영

5. **"템플릿 생성" 버튼 클릭**

### 검수 요청

1. **템플릿 목록에서 대기 상태 템플릿 확인**

2. **"검수 요청" 버튼 클릭**

3. **솔라피에서 검수 진행**
   - 검수 시간: 평균 1-2시간
   - 검수 결과: 승인 또는 반려

4. **검수 완료 후 승인 상태로 변경**

### 알림톡 발송

1. **승인된 템플릿 선택**

2. **"발송" 버튼 클릭**

3. **발송 정보 입력**:
   - 받는 사람 전화번호
   - 변수 값 (템플릿에 변수가 있는 경우)

4. **발송 미리보기 확인**
   - 변수가 치환된 실제 발송 내용 확인

5. **"알림톡 발송" 버튼 클릭**

## 📝 템플릿 작성 가이드

### 변수 사용법

템플릿 내용에 변수를 사용하려면 `#{변수명}` 형식을 사용하세요:

```
안녕하세요, #{이름}님!

주문하신 #{상품명}이(가) 배송 준비 중입니다.
배송 예정일: #{배송일}
```

발송 시 변수 값을 입력하면 자동으로 치환됩니다:
- `#{이름}` → "홍길동"
- `#{상품명}` → "노트북"
- `#{배송일}` → "2026-03-10"

### 버튼 추가

1. **웹링크 버튼**:
   - 버튼 이름: "자세히 보기"
   - 버튼 타입: WL (웹링크)
   - 모바일 링크: https://m.example.com
   - PC 링크: https://example.com

2. **배송조회 버튼**:
   - 버튼 이름: "배송조회"
   - 버튼 타입: DS (배송조회)

### 강조 표기형

특정 내용을 강조하려면 강조표기형을 사용하세요:

```
강조 제목: "특별 할인 안내"
강조 부제목: "50% 할인"
```

## 🔍 검수 통과 가이드

### 검수 기준

1. **정확한 정보**: 허위 정보나 과장 금지
2. **명확한 문구**: 이해하기 쉬운 문구 사용
3. **적절한 버튼**: 버튼 이름과 링크 일치
4. **광고 표시**: 광고성 메시지는 광고추가형 사용

### 반려 사유

일반적인 반려 사유:
- 부적절한 내용
- 불명확한 문구
- 버튼 링크 오류
- 광고성 메시지인데 기본형 사용
- 템플릿 이름과 내용 불일치

### 재검수 요청

반려된 경우:
1. 반려 사유 확인
2. 템플릿 수정
3. 다시 검수 요청

## 🛠️ API 엔드포인트

### 템플릿 관리

```typescript
// 카테고리 조회
GET /api/kakao/categories

// 템플릿 목록 조회
GET /api/kakao/templates?status=APPROVED

// 템플릿 단일 조회
GET /api/kakao/templates?templateId=xxxxx

// 템플릿 생성
POST /api/kakao/templates
Body: { name, content, channelId, categoryCode, ... }

// 템플릿 수정
PUT /api/kakao/templates
Body: { templateId, name, content, ... }

// 템플릿 삭제
DELETE /api/kakao/templates?templateId=xxxxx

// 검수 요청
POST /api/kakao/inspection
Body: { templateId, action: 'request' }

// 검수 취소
POST /api/kakao/inspection
Body: { templateId, action: 'cancel' }

// 알림톡 발송
POST /api/kakao/send
Body: { templateId, to, variables }
```

## 📊 상태 흐름도

```
[템플릿 생성]
     ↓
[PENDING (대기)]
     ↓
[검수 요청]
     ↓
[INSPECTING (검수중)]
     ↓
     ├─→ [APPROVED (승인)] → [발송 가능]
     └─→ [REJECTED (반려)] → [수정 후 재검수]
```

## ⚠️ 주의사항

1. **템플릿 내용 변경 불가**
   - 알림톡 발송 시 템플릿 내용은 변경할 수 없습니다.
   - 변수 값만 치환하여 발송됩니다.

2. **검수 완료 필수**
   - 검수가 완료된 템플릿만 발송할 수 있습니다.
   - 검수 시간: 평균 1-2시간

3. **채널 연동 필수**
   - 솔라피에 카카오 채널을 먼저 연동해야 합니다.
   - 채널 ID(pfId)를 정확히 입력해야 합니다.

4. **발신 번호 등록**
   - 솔라피에 발신 번호를 등록해야 합니다.
   - 환경 변수에 등록된 번호를 설정해야 합니다.

5. **과금 주의**
   - 알림톡 발송 시 비용이 발생합니다.
   - 솔라피 요금제를 확인하세요.
