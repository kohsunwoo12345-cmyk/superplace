
```
https://superplacestudy.pages.dev/dashboard/attendance-statistics/
```

### 3️⃣ 현재 달 확인
- ✅ 상단에 **"2026년 3월 출석일: X일"** 표시
- ✅ 달력에 출석 상태 표시 (🟢/🟡/🔴)
- ✅ 하단에 **출석 기록 목록** 표시

### 4️⃣ 이전 달로 이동
1. 우측 상단 **"이전 달"** 버튼 클릭
2. ✅ 달력이 이전 달로 변경
3. ✅ 출석 기록 목록도 이전 달 기록으로 변경
4. ✅ 상단 텍스트도 "2026년 2월 출석일: X일"로 변경

### 5️⃣ 다음 달로 이동
1. **"다음 달"** 버튼 클릭
2. ✅ 달력이 다음 달로 변경
3. ✅ 출석 기록 목록도 다음 달 기록으로 변경

### 6️⃣ 출석 기록 확인
- 각 기록에 다음 정보 표시:
  - 🟢/🟡/🔴 **상태 이모지**
  - **날짜** (03월 07일)
  - **요일** (목)
  - **전체 날짜** (2026-03-07)
  - **상태 배지** (출석/지각/결석)

### 각 기록 항목
```

│ 🟢  03월 07일 (목)         [출석]   │
```

- **왼쪽**: 상태 이모지 + 날짜 정보
- **오른쪽**: 상태 배지
- **호버 효과**: 배경색 변경

### 상태 배지 색상
- 🟢 **출석**: 초록색 (`bg-green-100 text-green-700`)
- 🟡 **지각**: 노란색 (`bg-yellow-100 text-yellow-700`)
- 🔴 **결석**: 빨간색 (`bg-red-100 text-red-700`)

## 💻 기술 구현

### 월 변경 함수
```typescript
const [currentMonth, setCurrentMonth] = useState<string>(() => {
  const now = new Date();
  return `${now.getFullYear()}-${String(now.getMonth() + 1).padStart(2, '0')}`;
});

const goToPreviousMonth = () => {
  const [year, month] = currentMonth.split('-').map(Number);
  const prevMonth = month === 1 ? 12 : month - 1;
  const prevYear = month === 1 ? year - 1 : year;
  setCurrentMonth(`${prevYear}-${String(prevMonth).padStart(2, '0')}`);
};

const goToNextMonth = () => {
  const [year, month] = currentMonth.split('-').map(Number);
  const nextMonth = month === 12 ? 1 : month + 1;
  const nextYear = month === 12 ? year + 1 : year;
  setCurrentMonth(`${nextYear}-${String(nextMonth).padStart(2, '0')}`);
};
```

### 출석 기록 필터링
```typescript
const monthRecords = Object.entries(calendarData)
  .filter(([date]) => date.startsWith(currentMonth))
  .map(([date, status]) => ({ date, status }))
  .sort((a, b) => b.date.localeCompare(a.date)); // 최신순
```
