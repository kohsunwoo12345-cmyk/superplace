# 출석 관리 수정 기능 완료

## 구현 내용

### ✅ 기능 개요
학원장과 선생님이 출석 관리 페이지에서 자동 출석 기록을 수동으로 수정할 수 있습니다.

### 📍 페이지 위치
```
/dashboard/attendance-management
```

### 🎯 주요 기능

#### 1. 출석 상태 수정 버튼 (3가지)
- **✅ 출석**: 정상 출석 처리 (녹색)
- **⏰ 지각**: 수업 시작 시간 이후 출석 (노란색)
- **❌ 결석**: 출석하지 않음 (빨간색)

#### 2. UI 개선 사항
- 버튼 크기 확대 및 아이콘 추가
- 현재 상태에 따라 활성/비활성 스타일 적용
- 색상으로 직관적인 상태 표시
- 상태 변경 시 명확한 성공 메시지

#### 3. 백엔드 API (이미 구현됨)

**POST** `/api/attendance/manual-update`

**Request Body:**
```json
{
  "userId": "student-id",
  "date": "2026-03-07",
  "status": "PRESENT" | "LATE" | "ABSENT",
  "updatedBy": "director-id"
}
```

**Response:**
```json
{
  "success": true,
  "message": "출석 상태가 업데이트되었습니다",
  "record": {
    "id": "attendance-xxx",
    "userId": "student-id",
    "userName": "홍길동",
    "date": "2026-03-07",
    "status": "PRESENT",
    "checkInTime": "2026-03-07 09:30:00"
  }
}
```

#### 4. 데이터베이스
- 테이블: `attendance_records_v3`
- 기존 기록이 있으면 UPDATE, 없으면 INSERT
- 한국 시간(KST) 자동 변환
- `updatedBy` 필드로 수동 수정 추적

### 📋 사용 시나리오

1. **학원장/선생님 로그인**
2. **출석 관리 페이지 접속** (`/dashboard/attendance-management`)
3. **날짜 선택** (기본값: 오늘)
4. **학생 목록 표시** (학원 소속 학생들)
5. **상태 버튼 클릭**으로 출석/지각/결석 변경
6. **즉시 반영** 및 성공 메시지 표시

### 🔒 권한
- **학원장 (DIRECTOR)**: 자신의 학원 학생들의 출석 수정 가능
- **선생님 (TEACHER)**: 자신의 학원 학생들의 출석 수정 가능
- **학생**: 접근 불가

### 💡 자동 출석 처리
- 매일 밤 11시에 출석하지 않은 학생은 자동으로 결석 처리
- 학원장/선생님이 언제든지 수정 가능

### 🎨 UI 스타일

#### 버튼 스타일
```tsx
// 활성 상태 (선택됨)
className="bg-green-600 hover:bg-green-700 text-white"

// 비활성 상태 (선택 안됨)
className="border-green-300 hover:bg-green-50 text-green-700"
```

#### 배지 색상
- 출석: 녹색 (`bg-green-500`)
- 지각: 노란색 (`bg-yellow-500`)
- 결석: 빨간색 (`bg-red-500`)
- 미처리: 회색 (`variant="outline"`)

### 📊 안내 메시지
```
📌 출석 관리 안내
• ✅ 출석: 정상 출석 처리 (수업 시간 내 출석)
• ⏰ 지각: 수업 시작 시간 이후 출석
• ❌ 결석: 출석하지 않음

💡 자동 처리: 매일 밤 11시에 출석하지 않은 학생은 자동으로 결석 처리됩니다
✏️ 수정 가능: 학원장과 선생님은 언제든지 자동 출석 기록을 수정할 수 있습니다
```
