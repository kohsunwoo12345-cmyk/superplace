### 2단계: 새 숙제 만들기
1. "새 숙제 내기" 버튼 클릭
2. 대상 학생 → "특정 학생" 선택

### 3단계: 즉시 나타나는 정보 확인

#### A. 알림창이 뜹니다
- ✅ 성공: "학생 5명 로드 완료!"
- ⚠️ 경고: "학생 데이터가 없습니다. 학생을 먼저 등록하세요."
- ❌ 오류: "API 오류: {상세 메시지}"

#### B. 화면에 표시되는 정보
```

```

## 🎯 예상 결과별 원인 파악

### 결과 1: "학생 5명 로드 완료!" + 학생 목록 표시
✅ **정상 작동!** 문제 없음

### 결과 2: "학생 데이터가 없습니다" + students.length = 0
원인: DB에 학생이 없음
해결: 학생 등록 필요

### 결과 3: "API 오류: 학원이 배정되지 않았습니다"
원인: academyId가 null
해결: 선생님/학원장에게 학원 배정 필요

### 결과 4: "API 오류: 인증이 필요합니다"
원인: 토큰이 없거나 유효하지 않음
해결: 로그아웃 후 재로그인

### 결과 5: 네트워크 오류
원인: API 호출 실패
해결: 서버 로그 확인 필요
