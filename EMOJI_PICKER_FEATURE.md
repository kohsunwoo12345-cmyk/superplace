# AI 봇 이모지 선택 기능

## 📋 개요

AI 봇 제작 시 **390개 이상의 이모지**를 **13개 카테고리**로 분류하여 쉽게 선택할 수 있는 드롭다운 선택기를 구현했습니다.

---

## 🎨 주요 기능

### 1. **이모지 선택기**
- ✅ **드롭다운 형식**: 클릭하면 팝업으로 이모지 목록 표시
- ✅ **실시간 미리보기**: 선택한 이모지가 즉시 표시
- ✅ **외부 클릭 시 닫힘**: 편리한 UX
- ✅ **선택 상태 표시**: 현재 선택된 이모지 하이라이트

### **2. State 관리**

```typescript
const [showEmojiPicker, setShowEmojiPicker] = useState(false);
const [formData, setFormData] = useState({
  icon: "🤖", // 기본 아이콘
  // ...
});

// 외부 클릭 시 닫기
useEffect(() => {
  const handleClickOutside = (e: MouseEvent) => {
    const target = e.target as HTMLElement;
    if (showEmojiPicker && !target.closest('.emoji-picker-container')) {
      setShowEmojiPicker(false);
    }
  };

  if (showEmojiPicker) {
    document.addEventListener('mousedown', handleClickOutside);
    return () => document.removeEventListener('mousedown', handleClickOutside);
  }
}, [showEmojiPicker]);
```

### **3. 이모지 선택 UI**

```tsx
<div className="relative emoji-picker-container">
  {/* 선택 버튼 */}
  <div 
    className="flex items-center gap-2 p-2 border rounded-lg cursor-pointer hover:bg-gray-50"
    onClick={() => setShowEmojiPicker(!showEmojiPicker)}
  >
    <span className="text-3xl">{formData.icon}</span>
    <span className="text-sm text-gray-500 flex-1">
      이모지 선택하기
    </span>
    <svg className="h-4 w-4 text-gray-400">
      {/* 화살표 아이콘 */}
    </svg>
  </div>
  
  {/* 이모지 팝업 */}
  {showEmojiPicker && (
    <div className="absolute z-50 mt-2 w-full max-h-96 overflow-y-auto bg-white border rounded-lg shadow-lg">
      {Object.entries(emojiCategories).map(([category, emojis]) => (
        <div key={category} className="p-3 border-b">
          <h4 className="text-xs font-semibold text-gray-700 mb-2">
            {category}
          </h4>
          <div className="grid grid-cols-8 gap-1">
            {emojis.map((emoji) => (
              <button
                key={emoji}
                type="button"
                className={`text-2xl p-2 rounded hover:bg-gray-100 ${
                  formData.icon === emoji ? 'bg-blue-100 ring-2 ring-blue-500' : ''
                }`}
                onClick={() => {
                  setFormData({ ...formData, icon: emoji });
                  setShowEmojiPicker(false);
                }}
              >
                {emoji}
              </button>
            ))}
          </div>
        </div>
      ))}
    </div>
  )}
</div>
```
