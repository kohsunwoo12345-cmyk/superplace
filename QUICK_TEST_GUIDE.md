## 📋 테스트 순서

### 1단계: 미승인 사용자 로그인 시도 (실패 확인)

1. 사이트 접속: https://3000-i9j6yk3jh6bv6fx7v3pje-2e77fc33.sandbox.novita.ai
2. "로그인" 클릭
3. 다음 정보 입력:
   - 이메일: `user@example.com`
   - 비밀번호: `user123!@#`
4. ❌ **예상 결과**: "관리자 승인 대기 중입니다. 승인 후 로그인하실 수 있습니다" 에러 메시지

### 2단계: 관리자 로그인

1. 다음 정보로 로그인:
   - 이메일: `admin@example.com`
   - 비밀번호: `admin123!@#`
2. ✅ **예상 결과**: 로그인 성공, 대시보드 표시

### 3단계: 승인 대기 사용자 확인

1. 관리자 대시보드에서 **"관리자 대시보드"** 메뉴 클릭
2. 상단에 **오렌지색 "승인 대기 중인 회원"** 카드 확인
3. 카드에 표시된 정보:
   - 👤 이름: 테스트 사용자
   - 📧 이메일: user@example.com
   - 🏢 회사: Test Company
   - 📱 전화: 010-1234-5678
   - 📅 가입일
   - 🏷️ "승인 대기" 배지 (오렌지색)

### 4단계: 사용자 승인

1. **"승인하기"** 버튼 클릭 (녹색 버튼, CheckCircle 아이콘)
2. 버튼이 "승인 중..." 로 변경되며 로딩 스피너 표시
3. ✅ **예상 결과**: 
   - "승인 완료" Toast 알림 표시
   - 승인 대기 목록에서 해당 사용자 제거됨

### 5단계: 승인된 사용자 로그인 (성공 확인)

1. 로그아웃
2. 다시 다음 정보로 로그인:
   - 이메일: `user@example.com`
   - 비밀번호: `user123!@#`
3. ✅ **예상 결과**: 로그인 성공, 사용자 대시보드 접근 가능

## 🔄 추가 테스트: 신규 회원가입

1. 로그아웃
2. "회원가입" 클릭
3. 새로운 정보 입력 (예: test@test.com)
4. ✅ **예상 결과**: "회원가입 신청이 완료되었습니다. 관리자 승인 후 로그인하실 수 있습니다." 메시지
5. 로그인 시도 → 승인 대기 에러
6. 관리자로 로그인 → 승인 대기 목록에 새 회원 표시
7. 승인 → 새 회원 로그인 가능
