# ⚡ 배포 명령어 빠른 답변

---

## 🎯 질문: "프로덕션 이외 분기 배포 명령"에 뭘 입력하나요?

---

## ✅ 답변: **비워두세요!**

```
프로덕션 이외 분기 배포 명령 (선택 사항)
[                                           ]
↑ 아무것도 입력하지 마세요!
```

---

## 📋 전체 입력 요약

### ✅ 입력해야 하는 것

**빌드 명령:**
```
npm run build
```

**빌드 출력 디렉토리:**
```
.next
```

### ❌ 입력하지 않아도 되는 것

**프로덕션 이외 분기 배포 명령:**
```
(비워둠)
```
> 선택 사항이므로 비워두면 자동으로 위의 빌드 명령 사용!

---

## 🔍 왜 비워두나요?

1. **선택 사항**입니다 (필수 아님)
2. 비워두면 **자동으로 빌드 명령 사용**
3. Next.js는 모든 브랜치에서 **같은 방식으로 빌드**
4. **설정이 더 간단**합니다

---

## ⚠️ 입력하면 안 되는 것

**이런 것들 입력하지 마세요:**

❌ `npx wrangler deploy`
❌ `npx wrangler pages deploy`
❌ `wrangler versions upload`

> 이것들은 로컬 CLI 명령어입니다!
> Cloudflare GUI에서는 사용하지 않습니다!

---
