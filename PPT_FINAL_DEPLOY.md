## 🔍 문제 해결

### 버튼이 안 눌린다면?
1. 브라우저 콘솔 열기 (F12)
2. 에러 메시지 확인
3. 페이지 새로고침 (Ctrl + F5)
4. 다른 브라우저에서 시도

### 라이브러리 로드 오류
- 알림: "PPT 라이브러리가 로드되지 않았습니다"
- 해결: 페이지 새로고침 후 재시도
- 인터넷 연결 확인

### 다운로드가 안 된다면?
- 브라우저 팝업 차단 해제
- Downloads 폴더 권한 확인
- 다른 브라우저에서 시도
