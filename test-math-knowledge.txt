# 수학 기초 개념 교재

## 1장: 2차 방정식

### 1.1 2차 방정식의 정의
2차 방정식은 ax² + bx + c = 0 형태의 방정식입니다. 여기서 a는 0이 아닌 실수입니다.

### 1.2 근의 공식
2차 방정식 ax² + bx + c = 0의 해는 다음 근의 공식으로 구할 수 있습니다:
x = (-b ± √(b²-4ac)) / 2a

### 1.3 판별식
판별식 D = b² - 4ac를 이용하여 근의 개수를 판단할 수 있습니다:
- D > 0: 서로 다른 두 실근
- D = 0: 중근 (한 개의 실근)
- D < 0: 서로 다른 두 허근

## 2장: 삼각함수

### 2.1 기본 삼각함수
직각삼각형에서:
- sin θ = 대변/빗변
- cos θ = 밑변/빗변
- tan θ = 대변/밑변

### 2.2 삼각함수의 기본 관계식
- sin²θ + cos²θ = 1
- tan θ = sin θ / cos θ
- 1 + tan²θ = sec²θ

### 2.3 삼각함수의 덧셈정리
- sin(A + B) = sin A cos B + cos A sin B
- cos(A + B) = cos A cos B - sin A sin B
- tan(A + B) = (tan A + tan B) / (1 - tan A tan B)

## 3장: 미적분

### 3.1 미분의 정의
함수 f(x)의 도함수 f'(x)는 다음과 같이 정의됩니다:
f'(x) = lim(h→0) [f(x+h) - f(x)] / h

### 3.2 기본 미분 공식
- (x^n)' = nx^(n-1)
- (sin x)' = cos x
- (cos x)' = -sin x
- (e^x)' = e^x
- (ln x)' = 1/x

### 3.3 적분의 정의
적분은 미분의 역연산입니다:
∫f(x)dx는 f(x)의 부정적분입니다.
F'(x) = f(x)일 때, ∫f(x)dx = F(x) + C (C는 적분상수)

### 3.4 미적분의 기본정리
∫[a,b] f(x)dx = F(b) - F(a)
여기서 F(x)는 f(x)의 부정적분입니다.
