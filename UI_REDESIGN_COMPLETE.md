## 🎯 새로운 기능

### 로고 클릭 → 대시보드 이동
```tsx
<Link href="/dashboard" className="...">
  <div className="w-12 h-12 bg-gradient-to-br from-blue-500 via-purple-600 to-pink-600 ...">
    <span>SP</span>
  </div>
  <div>
    SUPER PLACE AI
  </div>
</Link>
```

### 대시보드 버튼
```tsx
<Link href="/dashboard" className="px-4 py-2 bg-gradient-to-r from-blue-600 to-purple-600 ...">
  대시보드로 이동
</Link>
```

### Blob 애니메이션
```css
@keyframes blob {
  0% { transform: translate(0px, 0px) scale(1); }
  33% { transform: translate(30px, -50px) scale(1.1); }
  66% { transform: translate(-20px, 20px) scale(0.9); }
  100% { transform: translate(0px, 0px) scale(1); }
}
```

## 🎨 색상 팔레트

### 그라데이션
- **메인**: `from-blue-600 via-purple-600 to-pink-600`
- **배경**: `from-blue-50 via-purple-50 to-pink-50`
- **버튼**: `from-blue-600 to-purple-600`
- **로고**: `from-blue-500 via-purple-600 to-pink-600`

### 개별 색상
- **블루**: `blue-500`, `blue-600`
- **퍼플**: `purple-600`
- **핑크**: `pink-600`
- **노란색** (별점): `yellow-400`
- **그린**: `green-500` (통계)
