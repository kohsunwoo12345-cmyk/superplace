🔧 학원장 계정 localStorage 수정

📋 현재 상황

학원장 계정(wangholy1@naver.com)의 localStorage에 academyId가 없어서 AI 챗봇을 사용할 수 없는 상태입니다.

✅ 해결 방법

아래 버튼을 클릭하면 localStorage의 user 객체에 academyId를 추가합니다.

📝 수동 수정 방법

버튼이 작동하지 않으면 브라우저 콘솔(F12)에서 다음 명령어를 실행하세요:

const user = JSON.parse(localStorage.getItem('user'));
user.academyId = 'academy-1771479246368-5viyubmqk';
user.academyName = '꾸메땅학원';
localStorage.setItem('user', JSON.stringify(user));
console.log('✅ academyId 추가 완료!');
location.reload();
