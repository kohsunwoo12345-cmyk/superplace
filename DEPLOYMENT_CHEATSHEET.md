# 🎯 Cloudflare Pages 배포 - 빠른 참조 카드

## 📋 양식 입력값 (복사용)

### 기본 설정
```
프로젝트 이름: superplace_study
프레임워크: Next.js (드롭다운에서 선택)
브랜치: main
```

### 빌드 설정
```
빌드 명령: npm run build
출력 디렉토리: .next
루트 디렉토리: /
```

### 환경 변수 (3개)

**1. DATABASE_URL**
```
DATABASE_URL
file:./prisma/dev.db
```

**2. NEXTAUTH_URL**
```
NEXTAUTH_URL
https://superplace-study.pages.dev
```

**3. NEXTAUTH_SECRET**
```
NEXTAUTH_SECRET
[https://generate-secret.vercel.app/32 에서 생성]
```

1. ✅ Cloudflare 회원가입

4. ✅ "Save and Deploy" 클릭
5. ✅ 환경 변수 3개 추가
