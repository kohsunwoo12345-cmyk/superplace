**문제점**:
- `python-helper.ts`가 `/solve` 엔드포인트를 호출하려 함

**해결 방법 1: Worker에 /solve 엔드포인트 추가**
```javascript
// Worker 코드에 추가
if (url.pathname === "/solve" && request.method === "POST") {
  const body = await request.json();
  const { equation } = body;
  
  // 간단한 수학 계산 (eval 사용 주의)
  try {
    const result = eval(equation);
    return new Response(JSON.stringify({
      success: true,
      result: String(result),
      equation: equation
    }), { headers: corsHeaders });
  } catch (error) {
    return new Response(JSON.stringify({
      success: false,
      error: error.message
    }), { status: 400, headers: corsHeaders });
  }
}
```

**해결 방법 2: 기존 /grade 엔드포인트 활용**
