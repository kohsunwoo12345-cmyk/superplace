# 역할별 UI 개선 완료

## 📋 개요

관리자는 기존 시스템 관리 UI를 유지하고, 일반 사용자(학원장, 선생님, 학생)는 현대적이고 사용자 친화적인 UI를 제공합니다.

## 🎨 새로운 Modern UI 특징

### 1. **역할별 색상 테마**
- **학원장** 🏫: 인디고-퍼플 그라데이션 (`from-indigo-600 to-purple-600`)
- **선생님** 👨‍🏫: 블루-시안 그라데이션 (`from-blue-600 to-cyan-600`)
- **학생** 🎓: 그린-에메랄드 그라데이션 (`from-green-600 to-emerald-600`)

### 2. **현대적인 헤더 디자인**
- 깔끔한 화이트 헤더
- 그라데이션 로고 아이콘
- 역할별 대시보드 표시 (예: "학원장 대시보드")
- 검색 바 (데스크톱)
- 알림 아이콘 (빨간 점 표시)
- 사용자 프로필 아바타

### 3. **사이드바 개선**
- 현대적인 라운드 처리 (`rounded-xl`)
- 호버 시 그라데이션 배경 (`hover:from-blue-50 hover:to-purple-50`)
- 배지 표시 기능 (예: AI 튜터 - 1개의 새 메시지)
- 사용자 정보 카드
  - 역할별 아바타
  - 포인트 표시
  - 그라데이션 배경

### 4. **역할별 메뉴 구성**

#### 학원장 (DIRECTOR)
- 🏠 홈
- 👥 학생 관리
- 👨‍🏫 선생님 관리
- 📚 수업 관리
- ⏰ 출석 현황
- 📊 통계
- 💬 AI 도우미
- ⚙️ 설정

#### 선생님 (TEACHER)
- 🏠 홈
- 👥 내 학생들
- 📚 수업
- ⏰ 출석 체크
- 📝 숙제 확인
- 💬 AI 도우미
- ⚙️ 설정

#### 학생 (STUDENT)
- 🏠 홈
- 📚 내 수업
- 📝 숙제 제출
- ⏰ 출석 기록
- 💬 AI 튜터 (배지: 1)
- 🏆 성취도
- ⚙️ 설정

## 🔧 기술 구현

### 파일 구조
```
src/
├── components/
│   └── layouts/
│       └── ModernLayout.tsx      # 새로운 사용자 친화적 레이아웃
└── app/
    └── dashboard/
        └── layout.tsx             # 역할별 레이아웃 분기 처리
```

### 레이아웃 분기 로직
```typescript
// dashboard/layout.tsx
if (user && !isAdmin) {
  // 일반 사용자는 Modern Layout 사용
  return <ModernLayout role={user.role}>{children}</ModernLayout>;
}

// 관리자는 기존 레이아웃 계속 사용
// (드래그 앤 드롭, 메뉴 커스터마이징 기능 유지)
```

### 역할별 메뉴 생성
```typescript
const getMenuItems = (): MenuItem[] => {
  switch (role.toUpperCase()) {
    case 'DIRECTOR':
      return [/* 학원장 메뉴 */];
    case 'TEACHER':
      return [/* 선생님 메뉴 */];
    case 'STUDENT':
      return [/* 학생 메뉴 */];
    default:
      return [/* 기본 메뉴 */];
  }
};
```

## 📱 반응형 디자인

### 데스크톱 (lg 이상)
- 고정 사이드바 (w-64)
- 헤더 검색 바 표시
- 사용자 프로필 카드 표시

### 태블릿 (md ~ lg)
- 햄버거 메뉴로 전환
- 검색 바 숨김

### 모바일 (sm 이하)
- 전체 화면 슬라이드 사이드바
- 최대 너비 85vw
- 터치 친화적 버튼 크기

## 📊 UI 비교

### 헤더
| 항목 | 관리자 UI | 사용자 UI |
|------|----------|----------|
| 로고 | 텍스트만 | 그라데이션 아이콘 + 텍스트 |
| 검색 | 없음 | 검색 바 (데스크톱) |
| 알림 | 없음 | 알림 아이콘 + 배지 |
| 프로필 | 이메일 + 로그아웃 | 아바타 + 이름 + 로그아웃 |

### 사이드바
| 항목 | 관리자 UI | 사용자 UI |
|------|----------|----------|
| 배경 | 화이트 | 화이트 + 그라데이션 카드 |
| 메뉴 아이템 | 작은 사이즈 | 넓은 터치 영역 |
| 호버 효과 | 블루 배경 | 그라데이션 배경 |
| 특별 기능 | 드래그 앤 드롭 | 배지 표시 |
| 하단 | 드래그 가이드 | 사용자 프로필 카드 |

## 🎨 디자인 시스템

### 색상 팔레트
```css
/* 학원장 */
bg-gradient-to-br from-indigo-600 to-purple-600

/* 선생님 */
bg-gradient-to-br from-blue-600 to-cyan-600

/* 학생 */
bg-gradient-to-br from-green-600 to-emerald-600

/* 호버 효과 */
hover:bg-gradient-to-r hover:from-blue-50 hover:to-purple-50
```

### 라운드 처리
```css
/* 헤더 로고 */
rounded-xl

/* 메뉴 아이템 */
rounded-xl

/* 사이드바 카드 */
rounded-xl

/* 버튼 */
rounded-lg
```

### 그림자
```css
/* 헤더 */
shadow-sm

/* 로고 */
shadow-lg

/* 사이드바 */
shadow-sm

/* 모바일 메뉴 */
shadow-2xl
```
