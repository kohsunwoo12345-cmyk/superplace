## 🎯 기능 소개

카카오 알림톡 발송 페이지는 학원에서 학생/학부모에게 맞춤형 알림톡을 대량으로 발송할 수 있는 기능입니다.

### 주요 특징
- ✅ **5개 기본 템플릿** 제공 (Solapi 승인 완료)
- ✅ **3가지 수신자 입력 방식**: 학생 선택, 직접 입력, 엑셀 업로드
- ✅ **자동 URL 생성**: 각 학생마다 고유한 랜딩페이지 URL 자동 생성
- ✅ **실시간 미리보기**: 발송 전 메시지 내용 확인
- ✅ **변수 치환**: #{name}, #{url} 자동 치환

## 📱 발송 방법

### 1단계: 페이지 접속
```
https://superplacestudy.pages.dev/dashboard/kakao-alimtalk/send/
```
또는 대시보드 → 알림톡 관리 → **알림톡 발송하기** 클릭

### 2단계: 발송 설정

#### (1) 채널 선택 *필수
- 등록된 카카오 채널 중 선택
- 예: "꾸메땅학원"

#### (2) 템플릿 선택 *필수
기본 제공 템플릿 5개:
- **기본 템플릿 1**: 간단한 인사 메시지
- **기본 템플릿 2**: 안내 메시지
- **기본 템플릿 3 - 학습 안내**: 랜딩페이지 링크 포함 (추천)
- **기본 템플릿 4**: 짧은 링크 포함 메시지
- **기본 템플릿 5**: 짧은 링크 포함 메시지

**템플릿 3 예시:**
```
[학습 안내]

안녕하세요, #{name} 학생 학부모님
꾸메땅학원입니다.

오늘 준비된 맞춤형 학습 페이지 안내드립니다.
아래 링크를 클릭하여 이번달 리포트를 확인해 주세요!

■ 학습 페이지: #{url}

※ 본 메시지는 수신 동의하신 분들께 발송되는 학습 안내 정보입니다.
```

#### (3) 랜딩페이지 (선택)
- 랜딩페이지를 선택하면 각 학생마다 고유 URL이 자동 생성됩니다
- 선택 안 함도 가능 (링크가 없는 메시지 발송)

### 3단계: 수신자 입력

세 가지 방법 중 선택:

#### 방법 1: 학생 선택 (DB 연동) ⭐ 추천
- DB에 등록된 학생 목록에서 선택
- 체크박스로 여러 명 선택 가능
- 학생 ID 기반으로 고유 URL 생성

**사용 시나리오:**
- 특정 반 학생들에게 발송
- 월말 리포트 일괄 발송
- 정기적인 학습 안내

#### 방법 2: 직접 입력
- 이름과 전화번호를 직접 입력
- "수신자 추가" 버튼으로 여러 명 추가
- 소규모 발송에 적합

**입력 형식:**
- 이름: 홍길동
- 전화번호: 01012345678 (숫자만 입력)

#### 방법 3: 엑셀 업로드 ⭐ 대량 발송

**엑셀 파일 형식:**
| 학생이메일 | 학부모이름 | 학부모연락처 |
|-----------|----------|------------|
| student001@kumettang.com | 김영희 | 010-1234-5678 |
| student002@kumettang.com | 이철수 | 010-2345-6789 |
| student003@kumettang.com | 박민수 | 010-3456-7890 |

**처리 과정:**
1. 엑셀 파일 업로드
2. 시스템이 자동으로 학생 이메일로 DB 검색
3. 해당 학생의 최신 랜딩페이지 URL 자동 매칭
4. 처리 결과 요약 표시:
   ```
   ✅ 총 50명 중 45명 처리 완료
   
   ⚠️ 학생을 찾을 수 없음: 3명
   ⚠️ 리포트가 없음: 2명
   ```

### 4단계: 미리보기 확인
- 오른쪽 패널에 실제 발송될 메시지 미리보기
- 변수가 실제 데이터로 치환된 내용 표시
- #{name} → 학생 이름
- #{url} → 고유 랜딩페이지 URL

### 5단계: 발송
- **"X명에게 발송"** 버튼 클릭
- 발송 중 상태 표시
- 완료 시 성공 메시지

## 🎬 시나리오별 사용법

### 시나리오 1: 반별 월말 리포트 발송

**상황:** 수학 A반 학생 30명에게 월말 학습 리포트 발송

**절차:**
1. 채널: "꾸메땅학원" 선택
2. 템플릿: "기본 템플릿 3 - 학습 안내" 선택
3. 랜딩페이지: "2024년 3월 수학 리포트" 선택
4. **학생 선택** 탭 클릭
5. 수학 A반 학생 30명 체크박스 선택
6. 미리보기에서 메시지 확인:
   ```
   [학습 안내]
   
   안녕하세요, 홍길동 학생 학부모님
   꾸메땅학원입니다.
   ...
   ■ 학습 페이지: https://superplacestudy.pages.dev/landing/31?studentId=1&ref=...
   ```
7. **"30명에게 발송"** 클릭

**결과:**
- 각 학생의 학부모에게 개별 맞춤 메시지 발송
- 각 학생마다 고유 랜딩페이지 URL 포함

---

### 시나리오 2: 엑셀로 전체 학원생 대량 발송

**상황:** 전체 학원생 200명에게 여름방학 특강 안내

**준비:**
1. 엑셀 파일 준비 (sample_excel.xlsx):
   - 컬럼: 학생이메일, 학부모이름, 학부모연락처
   - 200행 데이터

**절차:**
1. 채널: "꾸메땅학원" 선택
2. 템플릿: "기본 템플릿 3 - 학습 안내" 선택
3. 랜딩페이지: "여름방학 특강 안내" 선택
4. **엑셀** 탭 클릭
5. 엑셀 파일 업로드 영역 클릭하여 파일 선택
6. 시스템 자동 처리 대기 (약 5-10초)
7. 처리 결과 확인:
   ```
   ✅ 총 200명 중 195명 처리 완료
   
   ⚠️ 학생을 찾을 수 없음: 3명
   ⚠️ 리포트가 없음: 2명
   
   각 학생의 최신 랜딩페이지 URL이 자동으로 설정되었습니다.
   ```
8. 미리보기에서 첫 번째 수신자 메시지 확인
9. **"195명에게 발송"** 클릭

**결과:**
- 195명에게 개별 맞춤 메시지 발송
- 각자 최신 랜딩페이지 URL 자동 삽입
- 실패한 5명은 별도 관리 필요

---

### 시나리오 3: 긴급 공지 직접 입력

**상황:** 내일 휴원 공지를 특정 학부모 3명에게 긴급 발송

**절차:**
1. 채널: "꾸메땅학원" 선택
2. 템플릿: "기본 템플릿 1" 선택 (링크 불필요)
3. 랜딩페이지: "선택 안 함"
4. **직접 입력** 탭 클릭
5. 첫 번째 수신자:
   - 이름: 김철수 학부모
   - 전화번호: 01011112222
6. "수신자 추가" 버튼 클릭
7. 두 번째, 세 번째 수신자도 동일하게 입력
8. 미리보기 확인
9. **"3명에게 발송"** 클릭

## 🔧 변수 사용법

### 사용 가능한 변수

| 변수 | 설명 | 예시 |
|-----|------|------|
| `#{name}` | 학생 이름 | 홍길동 |
| `#{이름}` | 학생 이름 | 홍길동 |
| `#{학생이름}` | 학생 이름 | 홍길동 |
| `#{url}` | 랜딩페이지 URL | https://superplacestudy.pages.dev/landing/31?... |
| `#{URL}` | 랜딩페이지 URL | https://superplacestudy.pages.dev/landing/31?... |
| `#{링크}` | 랜딩페이지 URL | https://superplacestudy.pages.dev/landing/31?... |
| `#{리포트URL}` | 랜딩페이지 URL | https://superplacestudy.pages.dev/landing/31?... |

### 변수 치환 예시

**템플릿:**
```
안녕하세요, #{name} 학부모님!
이번 달 학습 리포트를 확인해주세요.
링크: #{url}
```

**실제 발송:**
```
안녕하세요, 홍길동 학부모님!
이번 달 학습 리포트를 확인해주세요.
링크: https://superplacestudy.pages.dev/landing/31?studentId=1&studentEmail=...
```

## ❓ FAQ

### Q1: 학생 선택 탭에 학생이 안 보여요
**A:** 다음을 확인하세요:
- 로그인 상태 확인 (F12 → Console에서 `localStorage.getItem('token')` 확인)
- 학생이 DB에 등록되어 있는지 확인
- 학생의 전화번호가 입력되어 있는지 확인
- 페이지 새로고침 시도

### Q2: 템플릿 선택 시 미리보기가 안 나와요
**A:** 최신 배포 버전(ef345e20)에서 수정되었습니다. 페이지 새로고침 후:
- 템플릿만 선택해도 기본 내용이 미리보기에 표시됩니다
- 수신자 선택 전에는 `[수신자 이름]` 같은 플레이스홀더가 표시됩니다

### Q3: 엑셀 업로드 시 학생을 찾을 수 없다고 나와요
**A:** 엑셀의 학생이메일이 DB에 등록된 이메일과 정확히 일치해야 합니다:
- 대소문자 구분 확인
- 공백 제거
- 이메일 형식 확인 (예: student001@kumettang.com)

### Q4: 랜딩페이지를 선택하지 않으면 어떻게 되나요?
**A:** 랜딩페이지는 선택 사항입니다:
- 선택하지 않으면: #{url} 변수가 `[랜딩페이지 URL]`로 표시됨
- 링크가 필요 없는 단순 공지는 랜딩페이지 없이 발송 가능

### Q5: 발송 실패 시 어떻게 되나요?
**A:** 
- 실패한 건수와 이유가 에러 메시지로 표시됩니다
- 부분 성공 가능 (예: 50명 중 45명 성공, 5명 실패)
- 실패 사유: 잘못된 전화번호, API 오류 등

### Q6: 발송 비용은 어떻게 되나요?
**A:**
- Solapi API를 통해 발송되며, 건당 과금
- 정확한 비용은 Solapi 대시보드에서 확인
- 포인트 충전 필요

### Q7: 같은 메시지를 여러 번 발송해도 되나요?
**A:** 
- 네, 가능합니다
- 단, 카카오 정책상 스팸으로 간주될 수 있으니 주의
- 하루 발송량 제한 확인 필요

### Q8: 템플릿을 추가하고 싶어요
**A:**
1. 대시보드 → 알림톡 관리 → **템플릿 관리**
2. **새 템플릿 등록** 클릭
3. 템플릿 내용 작성
4. Solapi에서 승인 후 사용 가능
