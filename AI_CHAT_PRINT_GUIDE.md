# AI 챗 문제지 출력 - 학생용/강사용 분리 가이드

## 📋 개요

AI 챗에서 문제를 요청하면 **학생용 시험지**와 **강사용 해설지**를 자동으로 분리하여 한 번에 인쇄할 수 있습니다.

---

## ✅ 지원되는 포맷

### 1️⃣ 학생용/강사용 명시적 분리

AI가 아래와 같은 형식으로 응답하면 자동으로 인식됩니다:

```
**학생용 시험지**

1. 12를 소인수분해하면?
   ① 2² × 3
   ② 2 × 3²
   ...

**강사용 해설지**

**정답 및 해설**

1. 정답: ①
   해설: 12 = 2 × 6 = 2 × 2 × 3 = 2² × 3
   단계별 풀이: ...
```

### 2️⃣ 자동 구분 키워드

다음 키워드를 기준으로 문제와 답안을 자동 분리합니다:

- `**강사용`
- `**정답 및 해설**`
- `**정답**`
- `**해설**`
- `## 정답`
- `## 해설`

### 3️⃣ 답안 포맷

각 문제의 답안은 다음 형식을 지원합니다:

```
1. 정답: ①
   해설: 상세 설명...
   단계별 풀이: ...
   주의사항: ...
   팁: ...
```

## 🎯 사용 방법

### Step 1: AI에게 문제 요청

AI 챗에서 다음과 같이 요청하세요:

```
소인수분해 객관식 문제 5개 출제해줘.
학생용 시험지와 강사용 해설지를 따로 만들어줘.
```

또는:

```
중학교 1학년 수학 문제 10개 출제해줘.
답안은 정답과 해설을 포함해서 별도로 작성해줘.
```

### Step 2: AI 응답 확인

AI가 다음과 같은 형식으로 응답하는지 확인:

- ✅ 문제 번호가 있는가? (1., 2., 3. 등)
- ✅ 답안 섹션이 분리되어 있는가?
- ✅ 각 답안에 정답과 해설이 포함되어 있는가?

### Step 3: 메시지 선택 (선택사항)

- 여러 AI 응답이 있는 경우, **체크박스**를 클릭하여 출력할 메시지를 선택할 수 있습니다.
- 선택하지 않으면 모든 AI 응답이 포함됩니다.

### Step 4: 문제지 출력 버튼 클릭

우측 상단의 **"🖨️ 문제지 출력"** 버튼을 클릭합니다.

### Step 5: 인쇄 미리보기 확인

새 창에서 다음과 같이 표시됩니다:

#### 📄 페이지 1: 학생용 시험지

```
슈퍼플레이스 학원
학습 문제지

이름: __________  날짜: 2026. 3. 15.

【 객관식 문제 】

1. 12를 소인수분해하면?
   ① 2² × 3
   ② 2 × 3²
   ③ 2³ × 3
   ④ 2 × 6
   ⑤ 3 × 4

2. 18의 소인수는 몇 개인가?
   ...
```

#### 📄 페이지 2: 강사용 해설지

```
슈퍼플레이스 학원
답안지

【 객관식 정답 및 해설 】

1. 정답: ①
   해설: 12 = 2 × 6 = 2 × 2 × 3 = 2² × 3
   단계별 풀이: 12를 가장 작은 소수 2로 나눈 후, 6을 다시 2와 3으로 분해합니다.

2. 정답: ②
   해설: 18 = 2 × 3²이므로 소인수는 2와 3, 총 2개입니다.
   주의사항: 지수는 개수가 아니라 거듭제곱 횟수입니다.
   ...
```

### Step 6: 전체 인쇄 또는 선택 인쇄

- **전체 인쇄**: 학생용 + 강사용 모두 인쇄
- **선택 인쇄**: 인쇄 대화상자에서 페이지 범위 지정
  - 예: 페이지 1만 선택 → 학생용 시험지만 인쇄
  - 예: 페이지 2만 선택 → 강사용 해설지만 인쇄

## 🔧 문제 해결

### ⚠️ "출력할 문제를 찾을 수 없습니다" 에러

**원인:**
- AI 응답에 번호가 없음 (1., 2., 3. 등)
- 문제 형식이 지원되지 않음

**해결 방법:**

1. AI에게 **명시적으로 번호를 요청**하세요:
   ```
   문제를 1., 2., 3. 형식으로 번호를 매겨서 출제해줘.
   ```

2. 지원되는 번호 형식:
   - ✅ `1. 문제 내용`
   - ✅ `1) 문제 내용`
   - ✅ `**1.** 문제 내용`
   - ✅ `문제 1: 문제 내용`
   - ✅ `Q1. 문제 내용`

### ⚠️ 답안이 "정답 없음"으로 표시됨

**원인:**
- AI가 답안 섹션을 따로 제공하지 않음
- 답안 형식이 인식되지 않음

**해결 방법:**

1. AI에게 **답안과 해설을 별도로 요청**하세요:
   ```
   문제를 출제한 후, 답안과 해설을 별도 섹션으로 작성해줘.
   예시:
   
   **정답 및 해설**
   1. 정답: ①
      해설: 상세 설명...
   ```

2. 지원되는 답안 키워드:
   - `**강사용 해설지**`
   - `**정답 및 해설**`
   - `**정답**`
   - `## 정답`

### ⚠️ 일부 문제만 인쇄됨

**원인:**
- 문제 번호가 연속되지 않음 (1, 2, 5, 7 등)
- 문제 내용이 너무 짧거나 길어서 필터링됨

**해결 방법:**

1. 브라우저 콘솔(F12)을 열고 다음을 확인:
   ```
   📊 Total problems extracted: X
   ```

2. 추출된 문제 수가 예상보다 적으면:
   - AI에게 **연속 번호**로 재요청
   - 문제 내용이 **5자 이상, 2000자 이하**인지 확인

## 🎨 인쇄 설정 권장 사항

### 프린터 설정

- **용지 크기**: A4
- **방향**: 세로 (Portrait)
- **여백**: 기본 (상하 1.5cm, 좌우 1cm)
- **컬러**: 흑백 또는 컬러 (선택)

### 브라우저별 설정

#### Chrome / Edge
1. `Ctrl + P` (인쇄)
2. 대상: 프린터 선택
3. 페이지: 전체 또는 범위 지정
4. 색상: 흑백 / 컬러
5. 배율: 100%

#### Safari
1. `Cmd + P` (인쇄)
2. 프린터 선택
3. 페이지 범위 지정
4. 인쇄 실행

## 🚀 고급 기능

### 1. 메시지 선택 출력

여러 AI 응답 중 특정 메시지만 출력하려면:

1. 출력하고 싶은 **AI 응답 옆의 체크박스**를 클릭
2. 문제지 출력 버튼 클릭
3. 선택된 메시지만 인쇄 미리보기에 표시됨

### 2. 객관식/서술형 자동 분류

- **객관식**: ①②③④⑤ 또는 (1)(2)(3) 형식이 있으면 자동 인식
- **서술형**: 위 형식이 없으면 서술형으로 분류

### 3. 답안 공간 자동 추가

- **객관식**: 답안 공간 없음
- **서술형**: 2줄 답안 공간 자동 추가
