# 📄 PDF 업로드 및 RAG 사용 가이드

## 🐛 문제 해결: "PDF 파일을 읽을 수 없습니다" 오류

최신 업데이트로 다음 문제들을 해결했습니다:
- PDF.js Worker 로딩 실패
- 불명확한 에러 메시지
- PDF 파싱 오류 처리 부족

---

## ✅ PDF 업로드 방법

### 1. AI Gem 생성 페이지 접속
https://superplacestudy.pages.dev/dashboard/admin/ai-bots/create/

### 2. PDF 파일 준비
**권장 PDF:**
- 크기: 10MB 이하
- 페이지: 20페이지 이하
- 텍스트 기반 PDF (이미지 스캔 PDF 불가)
- 암호화되지 않은 PDF

### 3. 업로드 절차
1. **기본 정보 입력**
   ```
   Gem 이름: 예) 고등학교 수학 교재 봇
   설명: 예) 2024 수학 교재 기반 답변
   시스템 프롬프트: 예) 당신은 수학 전문 AI입니다. 교재 내용을 바탕으로 답변하세요.
   ```

2. **📄 지식 베이스 섹션**
   - 스크롤 아래로 이동
   - "파일 업로드" 버튼 클릭
   - PDF 파일 선택

3. **업로드 진행 확인**
   - F12 (개발자 도구) → Console 탭 열기
   - 다음 로그 확인:
     ```
     📁 파일 업로드 시작: example.pdf
     📄 PDF 파일 파싱 중...
     📚 PDF.js Worker 설정: https://cdnjs.cloudflare.com/...
       └─ ArrayBuffer 생성 완료
       └─ PDF 로딩 태스크 생성 완료
       └─ PDF 로드 완료: 10페이지
       └─ 페이지 1/10 파싱 완료
       ...
     ✅ PDF 파싱 완료: 총 5432자
     ✅ 파일 읽기 완료
     💾 Knowledge Base 업데이트 완료
     ```

4. **업로드 성공 확인**
   - "1개 파일이 성공적으로 업로드되었습니다" 팝업
   - 지식 베이스 텍스트 영역에 PDF 내용 표시

5. **AI Gem 생성**
   - "✨ AI Gem 생성" 버튼 클릭
   - "백그라운드에서 업로드 중..." 메시지
   - 자동으로 봇 목록 페이지로 이동

## 🔍 문제 해결

### ❌ 오류 1: "PDF 파일이 손상되었거나 올바른 PDF 형식이 아닙니다"

**원인:** PDF 파일이 손상됨

**해결:**
1. PDF를 다시 다운로드
2. PDF 뷰어(Adobe Reader 등)로 열어보기
3. 다른 PDF 파일로 시도

---

### ❌ 오류 2: "PDF 파일이 암호화되어 있습니다"

**원인:** PDF에 암호 보호가 설정됨

**해결:**
1. PDF 암호 제거:
   - Adobe Acrobat: 보안 → 암호 제거
   - 온라인 도구: iLovePDF, Smallpdf 등
2. 암호 제거된 PDF로 다시 시도

---

### ❌ 오류 3: "PDF에서 텍스트를 추출할 수 없습니다"

**원인:** 이미지 기반 PDF (스캔된 문서)

**해결:**
1. **OCR 적용**:
   - Adobe Acrobat: 도구 → 텍스트 인식 → 이 파일
   - 온라인 OCR: https://www.onlineocr.net/
2. 텍스트 검색 가능한 PDF로 변환 후 재시도

**또는 직접 입력:**
1. PDF 내용을 복사
2. "지식 베이스" 텍스트 영역에 직접 붙여넣기

---

### ❌ 오류 4: "PDF 처리 중 오류가 발생했습니다"

**원인:** PDF.js Worker 로딩 실패

**해결:**
1. **페이지 새로고침** (Ctrl+F5 또는 Cmd+Shift+R)
2. **브라우저 캐시 삭제**
3. **다른 브라우저 시도** (Chrome, Firefox 권장)
4. **시크릿/InPrivate 모드**로 시도

## ✅ 업로드 후 RAG 테스트

### 1. AI 채팅 페이지 접속
https://superplacestudy.pages.dev/ai-chat

### 2. 생성한 AI Gem 선택
왼쪽 사이드바에서 방금 만든 봇 클릭

### 3. PDF 내용 관련 질문
```
예시:
- "이 교재의 1장 내용을 요약해줘"
- "삼각함수 덧셈정리가 뭐야?"
- "2차 방정식 푸는 방법 알려줘"
```

### 4. RAG 작동 확인
**성공 예시:**
```
질문: "2차 방정식의 근의 공식은?"

AI 답변:
"지식 베이스에 따르면, 2차 방정식 ax² + bx + c = 0의 
근의 공식은 x = (-b ± √(b²-4ac)) / 2a 입니다."
```

✅ "지식 베이스에 따르면" → PDF 내용 참고 중
✅ 정확한 공식 인용 → RAG 작동 확인

## 📋 지원되는 파일 형식

| 형식 | 확장자 | 상태 |
|------|--------|------|
| PDF | .pdf | ✅ 지원 |
| 텍스트 | .txt | ✅ 지원 |
| Markdown | .md | ✅ 지원 |
| JSON | .json | ✅ 지원 |
| CSV | .csv | ✅ 지원 |
| HTML | .html | ✅ 지원 |
| XML | .xml | ✅ 지원 |

**제한사항:**
- 최대 파일 크기: 10MB
- 권장 PDF 페이지: 20페이지 이하
- 텍스트 기반 PDF만 지원 (이미지 PDF는 OCR 필요)

## 🎯 테스트용 샘플 PDF

간단한 테스트를 위해 텍스트 파일로 먼저 시도해보세요:

1. **텍스트 에디터**에서 새 파일 생성
2. 아래 내용 붙여넣기:
   ```
   # 수학 기초 개념
   
   ## 2차 방정식
   2차 방정식 ax² + bx + c = 0의 근의 공식:
   x = (-b ± √(b²-4ac)) / 2a
   
   ## 삼각함수
   - sin²θ + cos²θ = 1
   - tan θ = sin θ / cos θ
   ```
3. `test-math.txt`로 저장
4. 업로드 → AI Gem 생성 → 채팅 테스트
