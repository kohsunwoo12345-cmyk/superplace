## 🏫 학원명 자동 표시

### API 연동
```typescript
// API 호출: /api/user/profile
let academyName = '학원';
try {
  const token = localStorage.getItem('token');
  const response = await fetch('/api/user/profile', {
    headers: { Authorization: `Bearer ${token}` }
  });
  if (response.ok) {
    const data = await response.json();
    academyName = data.academyName || academyName;
  }
} catch (error) {
  console.error('Failed to fetch academy name:', error);
}
```

### 표시 위치
```html
<div class="header">
  <div class="academy-name">서울수학학원</div>
  <div class="worksheet-title">학습 문제지</div>
</div>
```

### 특징
- ✅ 자동 조회: API에서 실제 학원명 가져오기
- ✅ 고정 표시: 편집 불가능 (정확성 보장)
- ✅ 대체 텍스트: API 실패 시 "학원" 표시

### 핵심 CSS
```css
.problems-container {
  column-count: 2;              /* 2단 레이아웃 */
  column-gap: 20px;              /* 간격 20px */
  column-rule: 1px solid #e5e7eb; /* 구분선 */
}

.problem {
  margin-bottom: 20px;
  page-break-inside: avoid;      /* 인쇄 시 잘림 방지 */
  break-inside: avoid;           /* 브라우저 잘림 방지 */
}
```

### 장점
- 자동 균등 배분: 브라우저가 자동으로 문제를 2단에 분배
- 인쇄 최적화: page-break-inside로 문제 중간 잘림 방지
- 반응형: 화면 크기에 따라 자동 조정
- 간단한 구현: float/grid 없이 column-count만으로 구현

## 📋 사용 방법

```
사용자: "중학교 2학년 수학 문제 10개 출제해줘"
AI: 1. 다음을 계산하시오: ...
    2. x의 값을 구하시오: ...
    (10개 문제 생성)
```

### Step 2: 문제지 출력
```
1. 우측 상단 "문제지 출력" 버튼 클릭
2. 팝업 창 열림 (자동으로 학원명 표시)

4. "인쇄" 버튼 클릭
5. 프린터 선택 → 출력
```

### Step 3: 출력 결과 확인
```
✅ 학원명: 자동 표시 (예: "서울수학학원")

✅ 1페이지: 5-7문제 표시
```
