### 1. 쇼핑몰 제품 카드 표시 개선
**문제**: 한 페이지에 1개 제품만 표시, 카드 크기가 너무 큼

**해결**:
- 한 페이지에 **3개 제품**이 동시에 보이도록 수정
- 제품 카드 크기 축소 (패딩, 폰트, 이미지 비율 모두 감소)
- 그리드 레이아웃 적용: 데스크톱 3열, 태블릿 2열, 모바일 1열
- 슬라이더 로직을 페이지 단위로 변경
- 자동 슬라이드는 4개 이상일 때만 작동

## 🎯 사용자 테스트 가이드

### 쇼핑몰 테스트
1. https://superplacestudy.pages.dev/store 접속
2. 하드 리프레시 (Ctrl+Shift+R 또는 Cmd+Shift+R)
3. 한 페이지에 3개 제품 확인
4. 카드 크기가 작아진 것 확인
5. 화살표 또는 점을 클릭하여 페이지 이동

### 학생 추가 테스트
1. `/dashboard/students` 접속
2. "학생 추가" 버튼 클릭
3. 모든 필드 입력:
   - 이름, 이메일, 비밀번호 (필수)
   - 학교, 학년 (필수), 소속반
   - 연락처, 학부모 연락처
4. 학생 추가 후 상세 페이지에서 모든 정보 확인
