## ✅ 해결된 문제

**이전**: "출력할 문제를 찾을 수 없습니다"

**원인**: AI 응답에서 번호 패턴을 인식하지 못함

**해결**: 다양한 문제 번호 형식 지원

---

## 🎯 지원하는 문제 형식

### 추가 지원된 패턴
- `**1.**` (마크다운 볼드)
- `[1]` (대괄호)
- `문제 1:` (한글)
- `Q1.` (영문 Q)
- `Question 1:` (영문)

---

## 📝 사용 방법

1. AI에게 "수학 문제 3개 출제해줘" 요청
2. AI 응답 옆 체크박스 클릭 (선택)
3. "🖨️ 문제지 출력" 버튼 클릭
4. 새 창에서 인쇄

---

## 🐛 문제 발생 시

**"문제를 찾을 수 없습니다" 나오면**:
1. AI 응답에 번호가 있는지 확인 (1., 2., 등)
2. 문제 메시지를 체크했는지 확인
3. 브라우저 콘솔(F12) 확인

**해결**: AI에게 형식 통일 요청
```
"위 문제를 다음 형식으로:

1. 문제1
2. 문제2
3. 문제3

정답:
1. 답1
2. 답2
3. 답3"
```
